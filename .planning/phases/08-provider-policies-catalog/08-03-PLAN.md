**DeepSeek-specific notes:**

- Tier type: trial credits (5M tokens, 30-day expiry) -- NOT recurring
- No rate limits enforced (best-effort serving, delays under heavy load)
- Config snippet should use `createTokenLimit` for the credit-based allocation
- Note credits expire after 30 days
- Note that after credits, pay-as-you-go pricing is very cheap ($0.28/M input)
- Note English and Chinese interface
- Note 10-minute inference timeout under load

**Qwen-specific notes:**

- Tier type: trial credits (1M tokens, 90-day expiry, Singapore region ONLY)
- Requires Alibaba Cloud account (not just Qwen account)
- CRITICAL: Free quota only available in Singapore deployment region
- Note Chinese and English interface (some pages Chinese-only)
- Note dual-limit mechanism (RPM AND RPS enforced simultaneously)
- Use `@ai-sdk/openai-compatible` with base URL `https://dashscope-intl.aliyuncs.com/compatible-mode/v1`
- Config snippet should show the openai-compatible provider setup with baseURL
- Note "Free Quota Only" safety feature to prevent overage charges

**Cloudflare-specific notes:**

- Tier type: recurring (10K neurons/day), but pricing uses "neurons" not tokens
- Requires both `CLOUDFLARE_API_TOKEN` and `CLOUDFLARE_ACCOUNT_ID` (two env vars)
- Use REST API approach (not Workers binding) -- this is for server use, not Cloudflare Workers
- Use `@ai-sdk/openai-compatible` with base URL `https://api.cloudflare.com/client/v4/accounts/{ACCOUNT_ID}/ai/v1`
- Config snippet must show the openai-compatible provider setup with dynamic base URL
- Note the neuron-to-token conversion varies by model
- Note API token needs `Workers AI - Read` and `Workers AI - Edit` permissions

**NVIDIA-specific notes:**

- Tier type: trial credits (1000 credits, can request 4000 more)
- Credit system is opaque (token-to-credit conversion not publicly documented)
- 40 RPM limit for trial
- Use `@ai-sdk/openai-compatible` with base URL `https://integrate.api.nvidia.com/v1`
- Config snippet should use createRateLimit for RPM and note credits are tracked separately

**Cohere-specific notes:**

- Tier type: trial key (1000 calls/month, non-commercial use only)
- CRITICAL: Trial keys are explicitly NOT permitted for production or commercial use
- 20 RPM for Chat, 1000 calls/month shared across ALL endpoints
- Note the non-commercial restriction prominently

**GitHub-specific notes:**

- Requires GitHub account (free) and Personal Access Token with `models:read` scope
- Use `@github/models` dedicated package OR `@ai-sdk/openai-compatible` with base URL
- Base URL: `https://models.github.ai/inference`
- Note OLD endpoint `models.inference.ai.azure.com` deprecated July 2025
- Include limit tiers (Low models: 15 RPM/150 RPD, High models: 10 RPM/50 RPD)
- Note per-request token limits (8K input / 4K output)
- Note designed for prototyping, not production

## How Limits Work in llm-router

- **User-configured limits are primary.** You set limits in your config based on your provider's current quotas.
- **No limits = key always available.** If you configure no limits, the key is used until the provider rejects it.
- **Retry proxy is the safety net.** When a provider returns 429 (rate limit), the retry proxy rotates to the next available key.
- **Shipped defaults are OFF by default.** The `applyRegistryDefaults` config toggle (default: false) means no stale data affects routing.
