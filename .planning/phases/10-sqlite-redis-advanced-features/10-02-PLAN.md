Class `RedisStorage implements StorageBackend`:

Private constructor accepting `client: Redis, prefix: string`. Private `closed: boolean = false`.

Static async `create(options: RedisStorageOptions): Promise<RedisStorage>`:

1. Dynamic `import('ioredis')` in try/catch -- throw `LLMRouterError` with code `MISSING_PEER_DEPENDENCY` and message "ioredis is required for RedisStorage. Install it: npm install ioredis" on failure
2. Extract `prefix = options.prefix ?? 'llm-router:'`
3. Create client: if `options.connection` is string, `new IoRedis(options.connection)`; if object, `new IoRedis(options.connection as ConstructorParameters<typeof IoRedis>[0])`
4. Wait for connection readiness: `await new Promise<void>((resolve, reject) => { client.once('ready', () => resolve()); client.once('error', (err: Error) => reject(new LLMRouterError('Redis connection failed: ' + err.message, { code: 'REDIS_CONNECTION_ERROR', cause: err }))); })`
5. Log via debug and return new RedisStorage(client, prefix)

Private helper `makeKey(provider, keyIndex, windowType, periodKey): string`:

- Returns `${this.prefix}${provider}:${keyIndex}:${windowType}:${periodKey}`

Private helper `getTtlForWindow(window: TimeWindow): number` (in seconds):

- `per-minute` -> 120 (2x safety margin)
- `hourly` -> 7200 (2 hours)
- `daily` -> 172800 (2 days)
- `monthly` -> 5184000 (60 days)
- `rolling-30d` -> 172800 (2 days, per daily bucket)

Private helper `ensureOpen()`: throw if `closed === true`

**increment():**

1. `this.ensureOpen()`
2. Get periodKey via `getPeriodKey(window, Date.now())`
3. Build Redis key via `this.makeKey(provider, keyIndex, window.type, periodKey)`
4. Calculate TTL via `this.getTtlForWindow(window)`
5. Create pipeline: `this.client.pipeline()`
6. Pipeline commands:
   - `pipeline.hincrby(key, 'prompt_tokens', tokens.prompt)`
   - `pipeline.hincrby(key, 'completion_tokens', tokens.completion)`
   - `pipeline.hincrby(key, 'call_count', callCount ?? 0)`
   - `pipeline.expire(key, ttlSeconds)`
7. Execute pipeline: `const results = await pipeline.exec()`
8. Check results is not null, throw `LLMRouterError` with code `REDIS_ERROR` if null
9. Extract new values: `Number(results[0]?.[1]) || 0` for promptTokens, etc.
10. Return UsageRecord with the Redis key as `id` (deterministic -- same composite key = same id, satisfying contract test `record2.id === record1.id`)

**getUsage():**

1. `this.ensureOpen()`
2. Get periodKey and build Redis key
3. `const fields = await this.client.hgetall(key)`
4. Return `{ promptTokens: Number(fields['prompt_tokens']) || 0, completionTokens: Number(fields['completion_tokens']) || 0, totalTokens: (prompt + completion), callCount: Number(fields['call_count']) || 0 }`

**get(key):**

1. `this.ensureOpen()`
2. Use SCAN to find all keys matching `${this.prefix}${key}:*` pattern
3. For each found key, HGETALL to read values
4. Parse Redis key to extract provider, keyIndex, windowType, periodKey
5. Map to UsageRecord[] (reconstruct TimeWindow from windowType, set durationMs based on type)
6. Note: SCAN is non-blocking and cursor-based -- use `scanStream()` from ioredis or iterate with `scan` command

**put(record):**

1. `this.ensureOpen()`
2. Get periodKey from record.window and record.timestamp via `getPeriodKey(record.window, record.timestamp)`
3. Build Redis key
4. Use pipeline to HSET all fields (not HINCRBY -- put is absolute, not additive):
   - `pipeline.hset(key, 'prompt_tokens', record.promptTokens.toString())`
   - `pipeline.hset(key, 'completion_tokens', record.completionTokens.toString())`
   - `pipeline.hset(key, 'call_count', '0')` (put doesn't specify call count)
   - `pipeline.expire(key, ttlSeconds)`
5. Execute pipeline

**reset():**

1. `this.ensureOpen()`
2. Get periodKey and build Redis key
3. `await this.client.del(key)` -- delete the hash key entirely

**resetAll():**

1. `this.ensureOpen()`
2. Build pattern: if no args, `${this.prefix}*`; if provider only, `${this.prefix}${provider}:*`; if provider+keyIndex, `${this.prefix}${provider}:${keyIndex}:*`
3. Use SCAN to find matching keys, DEL in batches
4. Use `scanStream` with MATCH pattern for non-blocking iteration

**close():**

1. Set `closed = true`
2. Call `this.client.quit()` (graceful shutdown, waits for pending commands to complete, per ioredis docs)

Debug logging: Use `debug('llm-router:redis')` namespace for key operations.

import { describe, it, expect, afterAll } from 'vitest';
import { createStorageContractTests } from '../../tests/contracts/storage.contract.js';
import { RedisStorage } from './RedisStorage.js';

// Check if Redis is available before running tests
// Uses default localhost:6379 -- standard dev/CI Redis
let redisAvailable = false;
let testStorage: RedisStorage | null = null;

try {
  // Attempt connection with short timeout
  testStorage = await RedisStorage.create({
    connection: {
      host: '127.0.0.1',
      port: 6379,
      connectTimeout: 1000,
      maxRetriesPerRequest: 0,
      lazyConnect: false,
    },
    prefix: 'llm-router-test:',
  });
  redisAvailable = true;
  await testStorage.close();
  testStorage = null;
} catch {
  // Redis not available -- tests will be skipped
}

// Conditionally run contract tests
if (redisAvailable) {
  // Use unique prefix per test run to avoid contamination
  const testPrefix = `llm-router-test-${Date.now()}:`;

  createStorageContractTests('RedisStorage', async () => {
    const storage = await RedisStorage.create({
      connection: {
        host: '127.0.0.1',
        port: 6379,
        connectTimeout: 2000,
        maxRetriesPerRequest: 1,
      },
      prefix: testPrefix,
    });
    return storage;
  });

  // Cleanup: flush test keys after all tests
  afterAll(async () => {
    // Create a temporary connection to clean up test keys
    try {
      const cleanup = await RedisStorage.create({
        connection: {
          host: '127.0.0.1',
          port: 6379,
          connectTimeout: 1000,
          maxRetriesPerRequest: 0,
        },
        prefix: testPrefix,
      });
      await cleanup.resetAll(); // Deletes all keys with test prefix
      await cleanup.close();
    } catch {
      // Ignore cleanup errors
    }
  });
} else {
  describe('RedisStorage - StorageBackend contract', () => {
    it.skip('Redis not available -- skipping contract tests', () => {
      // This test is skipped when Redis is not running
      // To run: start Redis on localhost:6379
    });
  });
}
