**1. Create src/catalog/fetchers.ts:**

Two async functions: `fetchModelsDev()` and `fetchOpenRouter()`.

`fetchModelsDev()`:

- Fetch from `https://models.dev/api.json` (this returns TOML-based data as JSON).
- Use `AbortSignal.timeout(5000)` for 5s timeout.
- Parse response as JSON array. Each entry has model metadata fields.
- Use Zod `safeParse()` per entry to validate. Define a `modelsDevEntrySchema` that maps API fields to ModelMetadata. Skip invalid entries with `debug('llm-router:catalog')` log.
- Map API response to ModelMetadata[] format. Normalize pricing to per-1M-tokens. Set capabilities from API data where available (default all false if missing). Set status to 'active'. Set qualityTier to 'mid' as default (static snapshot overrides with accurate tiers).
- Handle the actual models.dev API response format: it may be an object with model entries keyed by model ID, or an array. Check the response shape and adapt. If the API is unreachable or returns unexpected format, throw (caller handles fallback).
- Return ModelMetadata[].

`fetchOpenRouter()`:

- Fetch from `https://openrouter.ai/api/v1/models` (public endpoint, no auth needed).
- Same 5s timeout pattern.
- Parse response JSON (has `data` array with model objects).
- OpenRouter model object has: `id`, `name`, `context_length`, `pricing` (per-token strings), `architecture` (with modality, tokenizer).
- Convert pricing from per-token strings to per-1M-tokens numbers: `parseFloat(pricing.prompt) * 1_000_000`.
- Set capabilities: infer `vision` from modality if available, default others to false.
- Set provider from model ID prefix (e.g., 'google/gemini-2.0-flash' -> provider: 'google').
- Map to ModelMetadata[] format. Return.

Both functions should throw on failure — the DefaultModelCatalog handles fallback logic.

**1. Create src/catalog/DefaultModelCatalog.ts:**

Implements `ModelCatalog` interface from `src/types/interfaces.ts`.

Constructor: `constructor(emitter: EventEmitter, options?: { fetchFn?: typeof fetch })`.

- Accept optional fetchFn for testing (default: global fetch).
- Initialize: `cache: Map<string, ModelMetadata> | null = null`, `cacheTimestamp: number = 0`, `inflightFetch: Promise<void> | null = null`, `abortController: AbortController | null = null`, `closed: boolean = false`.
- TTL constant: `CACHE_TTL_MS = 24 * 60 * 60 * 1000` (24 hours).

**Private method `ensureLoaded()`:**

- If cache exists and not expired, return immediately.
- If inflight fetch in progress, await it and return (deduplication).
- Otherwise call `doRefresh()`.

**Private method `doRefresh()`:**

- Set `this.inflightFetch` to the Promise of the actual refresh work.
- Create new AbortController, store as `this.abortController`.
- Try: call `fetchModelsDev()`, then `fetchOpenRouter()`.
- Merge: models.dev entries go into a Map<string, ModelMetadata> keyed by id. Then iterate OpenRouter entries — only add if id not already in map (OpenRouter fills gaps).
- Enrich quality tiers: load static-catalog.json, build a `tierMap: Map<string, QualityTierType>` from it. For each merged model, if tierMap has a tier for that id, use it. Otherwise keep the default 'mid'.
- Calculate diff from previous cache: modelsAdded, modelsRemoved, unchanged counts.
- Update cache and cacheTimestamp.
- Emit 'catalog:refreshed' event with source: 'live', diff counts, timestamp.
- Catch: if cache exists (stale), log warning and keep it. Emit 'catalog:refreshed' with source: 'cache'. If no cache, fall back to static snapshot: load static-catalog.json, populate cache. Emit with source: 'static'.
- Finally: clear inflightFetch to null, clear abortController.

**`getModel(modelId: string)`:**

- Call `ensureLoaded()`.
- Return `cache.get(modelId) ?? null`.

**`listModels(filter?: ModelListFilter)`:**

- Call `ensureLoaded()`.
- Get all values from cache.
- Apply filters:
  - `filter.provider`: match model.provider.
  - `filter.capabilities`: for each capability in filter that is true, model must also have it true.
  - `filter.qualityTier`: exact match.
  - `filter.maxPrice`: model.pricing !== null && model.pricing.promptPer1MTokens <= maxPrice. Also include free models (pricing.promptPer1MTokens === 0).
- Filter out models with status === 'deprecated'.
- Return filtered array.

**`getCapabilities(modelId: string)`:**

- Call `ensureLoaded()`.
- Get model from cache, return capabilities or null.

**`refresh()`:**

- If inflight, await existing inflight. Otherwise call doRefresh().

**`close()`:**

- Set closed = true.
- If abortController exists, call abort().
- Clear cache.
- Clear inflightFetch.

**Debug logging:** Use `debug('llm-router:catalog')` throughout. Log: fetch attempts, skip counts, cache hits, fallback to static, refresh timing.

**Static snapshot loading:** Use `JSON.parse(readFileSync(...))` or dynamic import to load `static-catalog.json`. Since this is a Node.js library, `readFileSync` from `node:fs` with `new URL('./static-catalog.json', import.meta.url)` is appropriate. Alternatively, use `import staticData from './static-catalog.json' with { type: 'json' }` — BUT since tsup bundles, use `createRequire(import.meta.url)` to load JSON at runtime. The simplest approach: read the file using `readFileSync` with `fileURLToPath(new URL('./static-catalog.json', import.meta.url))`.
