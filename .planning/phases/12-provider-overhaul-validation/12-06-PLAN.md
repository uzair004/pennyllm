**Step 1: Create `scripts/e2e-test.ts`**

Manual test script run with real keys from .env file. NOT part of CI.

```typescript
// scripts/e2e-test.ts
// Run: npx tsx scripts/e2e-test.ts
// Requires: .env with API keys for target providers

import { config } from 'dotenv';
config(); // Load .env

import { createRouter, defineConfig } from '../src/index.js';
import { generateText } from 'ai';
```

Test flow per provider:

1. Check if env var is set (e.g., `CEREBRAS_API_KEY`). Skip if missing.
2. Create router with that single provider configured.
3. Test `router.chat()` — call `generateText({ model: router.chat(), prompt: 'Say hello in one word' })`.
4. Verify: response received, usage tracked.
5. Test `router.model('provider/model')` — call with specific model.
6. Log results.

Also test chain fallback:

1. Configure 2+ providers.
2. Create router with `models: ['provider1/model1', 'provider2/model2']`.
3. Call `generateText({ model: router.chat() })`.
4. Log which model was resolved.

Report format:

```
[E2E] Testing 5 providers...
[E2E] cerebras: OK (llama-4-maverick, 142 tokens, 1.2s)

[E2E] groq:     OK (llama-4-maverick, 112 tokens, 0.3s)
```

```typescript
import { createRouter, defineConfig } from 'pennyllm';
import { generateText } from 'ai';

const router = await createRouter(
  defineConfig({
    providers: {
      cerebras: { keys: [process.env.CEREBRAS_API_KEY!], priority: 1 },
      google: { keys: [process.env.GOOGLE_GENERATIVE_AI_API_KEY!], priority: 2 },
      groq: { keys: [process.env.GROQ_API_KEY!], priority: 3 },
    },
  }),
);

// Automatic routing through model chain
const { text } = await generateText({
  model: router.chat(),
  prompt: 'Explain quantum computing',
});

// Filter by capability
const { text: reasoning } = await generateText({
  model: router.chat({ capabilities: ['reasoning'] }),
  prompt: 'Solve this logic puzzle...',
});

// Direct model access (bypasses chain)
const { text: specific } = await generateText({
  model: await router.wrapModel('google/gemini-2.5-flash'),
  prompt: 'Hello',
});

// Check chain status
const status = router.getStatus();
console.log(status.availableModels, 'models available');
```
