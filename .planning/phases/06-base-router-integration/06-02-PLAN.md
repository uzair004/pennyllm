Create a minimal end-to-end POC script at `scripts/poc-gemini.ts` that:

1. Imports `createRouter` from the package and `generateText` from `ai`
2. Reads `GOOGLE_GENERATIVE_AI_API_KEY` from environment (exit with helpful message if not set)
3. Creates a router with a single Google provider config containing one key
4. Calls `const model = await router.wrapModel('google/gemini-2.0-flash')`
5. Calls `const result = await generateText({ model, prompt: 'Say hello in exactly 5 words.' })`

7. Verifies that `result.usage.promptTokens > 0` and `result.usage.completionTokens > 0`
8. Verifies that router usage was recorded (getUsage returns non-zero totals)
9. Calls `router.close()` to clean up
10. Exits with code 0 on success, code 1 on any failure
