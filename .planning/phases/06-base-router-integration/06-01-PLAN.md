Factory function that returns `LanguageModelV1Middleware` with closure over router context.

```typescript
import type { LanguageModelV1Middleware, LanguageModelV1StreamPart } from 'ai';
import type { UsageTracker } from '../usage/UsageTracker.js';
import debugFactory from 'debug';

const debug = debugFactory('llm-router:middleware');
```

Export `createRouterMiddleware(options: { provider: string; keyIndex: number; model: string; tracker: UsageTracker; requestId: string }): LanguageModelV1Middleware`

The returned middleware object:

**wrapGenerate:** `async ({ doGenerate }) => { ... }`

- Call `const result = await doGenerate()`
- Extract usage: `result.usage` has `{ promptTokens: number; completionTokens: number }`
- Fire-and-forget record: `tracker.record(provider, keyIndex, { promptTokens: result.usage.promptTokens, completionTokens: result.usage.completionTokens }, requestId, null).catch(err => debug('Usage recording failed: %s', err instanceof Error ? err.message : String(err)))`
- Return `result`

**wrapStream:** `async ({ doStream }) => { ... }`

This is the critical streaming implementation. The doStream() return type has shape `{ stream: ReadableStream<LanguageModelV1StreamPart>, rawCall, rawResponse, request, warnings }`.

Concrete implementation steps:

1. Call `const result = await doStream()` to get the original stream result.
2. Create a `TransformStream<LanguageModelV1StreamPart, LanguageModelV1StreamPart>` that intercepts the finish chunk:

```typescript
const transform = new TransformStream<LanguageModelV1StreamPart, LanguageModelV1StreamPart>({
  transform(chunk, controller) {
    // Pass every chunk through unmodified
    controller.enqueue(chunk);

    // When we see the 'finish' chunk, extract usage and record it
    if (chunk.type === 'finish') {
      const usage = chunk.usage; // { promptTokens: number; completionTokens: number }
      tracker
        .record(
          provider,
          keyIndex,
          { promptTokens: usage.promptTokens, completionTokens: usage.completionTokens },
          requestId,
          null,
        )
        .catch((err) =>
          debug(
            'Stream usage recording failed: %s',
            err instanceof Error ? err.message : String(err),
          ),
        );
    }
  },
});
```

3. Pipe the original stream through the transform: `const transformedStream = result.stream.pipeThrough(transform);`
4. Return the result with the transformed stream: `return { ...result, stream: transformedStream };`

This approach:

- Passes ALL stream chunks through unmodified (text-delta, tool-call, reasoning, etc.)
- Only intercepts the `finish` chunk (which has `type: 'finish'` with `usage: { promptTokens, completionTokens }`)
- Records usage fire-and-forget (never blocks the stream, never throws to user)
- Uses standard Web Streams API (TransformStream) which TypeScript and Node.js natively support

**Important:** Never throw from middleware. All errors from usage recording must be caught and logged. The user's LLM call must not fail because of router internals.

import { wrapLanguageModel } from 'ai';

Export `async function routerModel(router: Router, modelId: string, options?: { strategy?: string; estimatedTokens?: number; requestId?: string; registry?: ProviderRegistry }): Promise<LanguageModelV1>`

Implementation:

1. Call `const selection = await router.model(modelId, options)` to get `{ keyIndex, key, reason }`
2. Parse provider from modelId: `const provider = modelId.substring(0, modelId.indexOf('/'))` and `const modelName = modelId.substring(modelId.indexOf('/') + 1)`
3. Get or create registry: `const registry = options?.registry ?? await ProviderRegistry.createDefault()`
4. Create provider model: `const baseModel = createProviderInstance(registry, provider, modelName, selection.key)`
5. Generate requestId if not provided: `const requestId = options?.requestId ?? crypto.randomUUID()`
6. Create middleware: `const middleware = createRouterMiddleware({ provider, keyIndex: selection.keyIndex, model: modelName, tracker: router.usage, requestId })`
7. Return `wrapLanguageModel({ model: baseModel, middleware, modelId, providerId: 'llm-router' })`

Also export a `createModelWrapper(router: Router, registry?: ProviderRegistry)` convenience that returns a curried function `(modelId: string, options?) => Promise<LanguageModelV1>`. This avoids passing router on every call:

```typescript
export function createModelWrapper(router: Router, registry?: ProviderRegistry) {
  let resolvedRegistry: ProviderRegistry | undefined = registry;
  return async (
    modelId: string,
    options?: { strategy?: string; estimatedTokens?: number; requestId?: string },
  ): Promise<LanguageModelV1> => {
    if (!resolvedRegistry) {
      resolvedRegistry = await ProviderRegistry.createDefault();
    }
    return routerModel(router, modelId, { ...options, registry: resolvedRegistry });
  };
}
```
