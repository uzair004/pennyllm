**1. Create `src/storage/MemoryStorage.ts`** — Default in-memory StorageBackend implementation.

- Class `MemoryStorage` implements `StorageBackend` from `../types/interfaces.js`
- Internal `Map<string, UsageRecord>` for counter storage
- Internal `closed: boolean` flag, starts `false`
- Constructor emits stderr warning: `process.stderr.write('Warning: Using in-memory storage — usage data will not persist across restarts. Use a storage adapter for persistence.\n')`
- Use `debug('llm-router:storage')` for operational logging

**Composite key format:**

```
private makeKey(provider: string, keyIndex: number, window: TimeWindow): string {
  const period = Math.floor(Date.now() / window.durationMs);
  return `${provider}:${keyIndex}:${window.type}:${period}`;
}
```

**Lazy expiration cleanup** — `private cleanupExpired(window: TimeWindow)`:

- Compute `currentPeriod = Math.floor(Date.now() / window.durationMs)`
- Iterate over Map entries, delete any where `record.window.type === window.type` AND `Math.floor(record.timestamp / window.durationMs) < currentPeriod`
- Only clean entries matching the given window type (not full Map scan across all types)

**Every public method must:**

- Check `if (this.closed) throw new LLMRouterError('Storage backend is closed', { code: 'STORAGE_CLOSED' })` as first line
- Use `LLMRouterError` (not plain Error) from `../errors/base.js`

**Method implementations:**

- `get(key)`: Filter Map entries where mapKey starts with `${key}:`, return matching UsageRecord array
- `put(record)`: Set directly using `makeKey(record.provider, record.keyIndex, record.window)`
- `increment(provider, keyIndex, tokens, window)`: Call `cleanupExpired(window)`, then synchronous read-modify-write on Map (NO await between get and set). Create new record with `randomUUID()` id if not exists, else spread existing and add token values. Set `timestamp: Date.now()` on new records. `callCount` is NOT a field on UsageRecord — only track `promptTokens`, `completionTokens`, `totalTokens`.
- `getUsage(provider, keyIndex, window)`: Call `cleanupExpired(window)`, then get by key and return `record?.totalTokens ?? 0`
- `reset(provider, keyIndex, window)`: Delete by key
- `close()`: Set `this.closed = true`, then `this.data.clear()`
