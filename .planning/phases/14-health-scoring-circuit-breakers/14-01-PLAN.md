**Constants:**

```typescript
const WINDOW_SIZE = 10;
const HEALTH_THRESHOLD = 30; // circuit trips below 30%
const COOLDOWN_SCHEDULE_MS = [30_000, 60_000, 120_000, 300_000, 900_000];
```

**Class: HealthScorer**

- Constructor: `constructor(emitter: EventEmitter)` -- stores emitter reference. Uses `debugFactory('pennyllm:health')` for debug logging
- Private Maps: `windows: Map<string, RollingWindow>` and `circuits: Map<string, CircuitInfo>`

**Private helpers:**

- `getOrCreateWindow(provider: string): RollingWindow` -- returns existing or creates new window with `{ buffer: new Array(WINDOW_SIZE), index: 0, count: 0 }`. NOTE: do NOT pre-fill buffer with true. Use `count` to track actual outcomes
- `getOrCreateCircuit(provider: string): CircuitInfo` -- returns existing or creates `{ state: 'closed', cooldownUntil: 0, escalationLevel: 0, openedAt: 0, probeInFlight: false }`
- `calculateSuccessRate(window: RollingWindow): number` -- if count === 0 return 100. Otherwise count successes in `buffer.slice(0, window.count)` (or full buffer if count >= WINDOW_SIZE), return `(successes / relevantLength) * 100`
- `getCooldownMs(escalationLevel: number): number` -- return `COOLDOWN_SCHEDULE_MS[Math.min(escalationLevel, COOLDOWN_SCHEDULE_MS.length - 1)]!`
- `openCircuit(provider: string, circuit: CircuitInfo): void` -- set state to 'open', cooldownUntil to `Date.now() + getCooldownMs(circuit.escalationLevel)`, openedAt to Date.now() if not already set, probeInFlight to false. Log via debug: `'%s: circuit OPEN (health %d%%, threshold %d%%)'`
- `resetCircuit(provider: string, circuit: CircuitInfo): void` -- full reset: state='closed', escalationLevel=0, cooldownUntil=0, probeInFlight=false. Clear and reset the rolling window for this provider (fresh window). Log: `'%s: circuit CLOSED (recovered)'`

**Public methods:**

1. `shouldSkip(provider: string): SkipResult`
   - Get circuit for provider. If state is 'closed', return `{ skip: false }`
   - If state is 'open': check if `Date.now() >= circuit.cooldownUntil`. If no (still cooling), return `{ skip: true, reason: 'circuit_open_cooldown' }`. If yes (cooldown expired), transition to 'half-open', log `'%s: circuit HALF-OPEN (cooldown expired, probing)'`, set probeInFlight=true, return `{ skip: false }` (allow probe)
   - If state is 'half-open': if probeInFlight is true, return `{ skip: true, reason: 'circuit_half_open_probe_in_flight' }`. Otherwise return `{ skip: true, reason: 'circuit_half_open' }`

2. `recordSuccess(provider: string): void`
   - Record true in rolling window for provider
   - Get circuit. If state is 'half-open': emit `RouterEvent.PROVIDER_RECOVERED` with `{ provider, downtimeMs: Date.now() - circuit.openedAt, previousState: 'half-open', recoveredAt: new Date().toISOString(), timestamp: Date.now() }` via `safeEmit`. Then call `resetCircuit(provider, circuit)`. Log: `'%s: half-open probe SUCCESS, circuit CLOSED'`
   - If state is 'closed': check if success rate is above threshold (no-op, circuit stays closed)

3. `recordFailure(provider: string): void`
   - Record false in rolling window for provider
   - Get circuit. If state is 'half-open': escalate -- increment escalationLevel, call openCircuit. Log: `'%s: half-open probe FAILED, circuit OPEN (escalation %d)'`
   - If state is 'closed': calculate success rate. If below HEALTH_THRESHOLD, call openCircuit. Log: `'%s: health degraded to %d%%, circuit OPEN'`

4. `getHealthScore(provider: string): number`
   - If no window exists for provider, return 100 (cold start)
   - Return `calculateSuccessRate(window)`

5. `getCircuitState(provider: string): CircuitState`
   - If no circuit exists for provider, return 'closed'
   - Return circuit.state

6. `forceNearestHalfOpen(providers: string[]): string | null`
   - For the "all circuits open" fallback. Scan all provided providers. Find the one whose circuit has the smallest `cooldownUntil` that is finite (not 0 and not the initial state). Force it to 'half-open' and set probeInFlight=true. Return the provider string, or null if none found
   - Log: `'%s: forced to HALF-OPEN (all circuits open fallback)'`
